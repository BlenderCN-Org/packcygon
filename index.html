Packcygon

Packcygon applies a 2d bin-packing algorithm to selected objects, trying to place them within the smallest possible footprint.

Objects must have dimension along X and Y, thus empties, cameras and lamps are not considered.

An adjustible "fudge factor" fine-tunes the guess for target footprint size.

The downloaded zip can be installed via Blender's user-preferences.

There are two operators:

  • cygon pack
  • cygon packmodal

The non-modal one just uses a preset default guess.

The modal one lets you adjust the amount of guess with the mouse wheel.

Too much guess, not enough guess, negative fudge-factor? Then the output will look like an ell-shape or just a scattered mess. Things may end up off in some crazy direction. You may have to undo and retry. You may have to zoom out.

You may have to repeat multiple times if you want to make it through thousands of objects.

Apply rotation and scale to the objects. Clear the positions of the objects. Select the objects. Press the spacebar. Type cyg...

If using modal, wheel up for more fudge. Wheel down for less fudge. Left mouse, right-mouse, escape or return to be done with it.

No height. This is 2d bin-packing. Maybe someone will extend this into the third dimension. That probably will not be me.
